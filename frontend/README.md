## Requirements

For development, you will only need Node.js installed on your environment.

## Getting Started

```bash
npm i
npm dev
```

## Build for development

```bash
npm run dev
```

in another command line start the mock API Service

```bash
node ..\API-Mock\server.js
```

Open [http://localhost:3000](http://localhost:3000) to see dev server.

## Build for production

```bash
npm build
npm serve # to preview production build
```

## Tools

### Typescript

- [nextjs](https://nextjs.org/) is used as frontend library.
- [fontawesome](https://fontawesome.com/) is used as Icon Library.
- [formik](https://formik.org/) is used to handle forms in react.
- [yup](https://www.npmjs.com/package/yup) is used to handle form validation in react.
- [jest](https://jestjs.io/) is used for unit and component testing.

### CSS

- [TailwindCss](https://tailwindcss.com/) is used as Css Utility Framework
- [DaisyUI](https://daisyui.com/) extends TailwindCSS with ready UI components.
